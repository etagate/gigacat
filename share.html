1.2 Publish the file on IPFS

2. Share a directory of lists (medium)


2.1 Create a list of CIDs


Create a file.txt and write in CIDs of list you want to share (lists created following the instructions above) 

QmRhxBzxtqSbcSFj1RtFMRjFF9iUnVPE7bmzpaLQgzMRMH
QmPft65bHUEv9WjzUP6i3wqjRqvo1VtJvCnhzbPKELsKgv
Qmbr1TEC1eUXWHT9LxBTgCKra4F2nuh6m4UMrKaPbeYeBc


2.2 Publish the file on IPFS


$ ipfs add file.txt
added Qma8vdZMTT58BS4HqAcgsCRLzaaHHD6ELRKbSBBHUZnGdL file.txt
100 B / 100 B [=====================================] 100.00%

$ ipfs pin add Qma8vdZMTT58BS4HqAcgsCRLzaaHHD6ELRKbSBBHUZnGdL
pinned Qma8vdZMTT58BS4HqAcgsCRLzaaHHD6ELRKbSBBHUZnGdL recursively

2.3 Share file CID inside the link


add ?dir= at the end of the website URL together with the file CID 

https://giga.cat/?dir=Qma8vdZMTT58BS4HqAcgsCRLzaaHHD6ELRKbSBBHUZnGdL
